サンプルの実行

Windows

ダウンロードしたファイルを解凍したら、コマンドプロンプトを起動してpita.exeのあるディレクトリへ移動してください。 そして、以下のコマンドを入力し、SVGファイルが出力されればインストールは成功です。

...\pita-0.5.0-alpha-x86-win>pita.exe examples/3rects.cgl -o 3rects.svg

ソースコードのビルド

Pita 0.5.0 のビルドに必要な要件は以下の通りです。

  • Boost 1.54
  • CMake 3.1
  • C++14をサポートするコンパイラ

次のようにしてビルドすることができます。

git clone --recursive https://github.com/agehama/Pita.git
mkdir build
cd build
cmake ../Pita
make -j
